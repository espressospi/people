# 피플

한 사람의 작은 일상을 지켜보는 방치형 웹게임의 초기 테스트 버전이다.

## 실행

```bash
pnpm install
pnpm dev
```

브라우저에서 `http://127.0.0.1:5173`을 연다.

## 현재 테스트 흐름

1. 피플의 기본 정보와 행동 방침을 정한다.
2. 직접 행동하거나 자동 생활을 시작한다.
3. 오전 7시부터 오후 11시까지 행동한 뒤 한 살을 먹는다.
4. 모든 결과는 `data/saves/local-player.json`에 자동 저장된다.

초기 테스트 버전의 결정과 제약은 `.agent/INITIAL_TEST_VERSION.md`에 기록한다.
